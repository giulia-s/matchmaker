Matchmaker : A MATLAB tool for manual matching of (palaeoclimate) time series
=========================================
> Matchmaker is a collection of MATLAB scripts that form an application aiming at facilitating efficient and quality-checked matching of time series from different records. 

> It has been designed for use with ice core profiles, and some design choices clearly reflect this, but can essentially be used with any type of time-series data, although the tool is probably not suited for working with data sets from many independent cores/sites due to the screen layout.

**Refer to the User's guide pdf**, provided in this release, before starting to use the program.

**Please cite the original release of the program:** Rasmussen, S. et al.: Synchronization of the NGRIP, GRIP, and GISP2 ice cores across MIS 2 and palaeoclimatic implications, Quaternary Sci. Rev., 27, 18–28, https://doi.org/10.1016/j.quascirev.2007.01.016, 2008.

Matchmaker requires a MATLAB licence and has been tested on **MATLAB R2020b and R2024b**. 

To open in **Matlab Online**: https://matlab.mathworks.com/open/github/v1?repo=giulia-s/matchmaker
